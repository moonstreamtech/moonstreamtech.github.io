About the game

Last Tile is an English-language merge + survival game where every merge is a turn. The board starts as a small active region inside a locked frame and keeps growing as you climb tiers. Slides and swaps are free, splits recover from lockouts, and the run ends when no adjacent matching pair remains.

Status: Coming soon to Google Play.

Core Features

  • Growing board: 5×5 active → 7×7 → 9×9 … inside a 7×7 viewport with edge pan
  • Four deliberate actions: tap-tap merge, drag-to-merge, double-tap split, one-step slide/swap
  • Value-tier frame unlocks that open 1–6 random cells at a time
  • Run autosave — resume the exact same board after a cold restart
  • On-device top-10 leaderboard
  • Offline-first single-player design, no account required
