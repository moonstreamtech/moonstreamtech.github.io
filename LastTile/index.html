About the game

Last Tile is a single-player puzzle where every merge is a turn. The board starts as a small active region inside a locked frame and keeps growing as you climb tiers. Slides and swaps are free, splits recover from lockouts, and the run ends when no adjacent matching pair remains.

It is designed to be calm and focused — no timers, no ads, no account, no internet required.

Core features

  • Growing board. 5×5 active region opens up to 7×7, 9×9 and beyond as you reach value-tier milestones.
  • Four deliberate actions. Tap-tap merge, drag-to-merge, double-tap split, and one-step slide or swap.
  • Frame unlocks. Hitting a milestone tile opens new cells around the active area.
  • Hazards. Fire, ice and poison can infect a tile for a few turns. Burn them off with a high-tier merge nearby, or wait them out — the tile returns when the hazard fades.
  • Combo chain. Each merge adds to your multiplier; a non-matching tap or a slide resets it.
  • Run autosave. Resume the exact same board after closing the app.
  • On-device leaderboard. Your top ten runs, stored locally.
  • Offline-first. No account required, no data collected, no internet needed.

How to play

  1. Two adjacent tiles with the same value can be merged. The result is the next tier (e.g. two 4s become an 8).
  2. Each merge counts as a turn; a new tile spawns somewhere on the board.
  3. Reach milestone values to unlock additional cells and eventually grow the active region.
  4. If no two adjacent tiles match anywhere on the board, the run ends.
